ASSIGNMENT
==========
A program is needed to prompt the user operator for ten exam scores,
calculate the average score and display the result to the screen.
The average score is calculated as the sum of the scores divided by the number of scores.

VARIABLES
=========
integer int01
integer int02
integer int03
integer int04
integer int05
integer int06
integer int07
integer int08
integer int09
integer int09
integer intSum
integer intCount
float fResult

ALGORITHM
=========
define integers int01, int02, int03, int04, int05, int06, int07, int08, int09, int10, to hold input values
define integers intSum, intCount for use in calculation
define float fResult to hold result of average calculation
set intCount to 1
while intCount is less than 11 (ask for more input)
    ask user for input
    store the input in appropriate variable
    increment intCount
endwhile
add int01 + int02 + int03 + int04 + int05 + int06 + int07 + int08 + int09 + int10 and store the result in intSum
set fResult = intSum / 10 to calculate the average of the 10 values
display the average is intResult
